# draft-prospects

This folder is reserved for JSON/data and optional images for NBA draft prospects.

In the current implementation, the game builds its prospect list by importing your existing draft dataset from the sibling `Nba draft/` project at runtime (so your original data remains untouched).

If you want, I can also generate a real JSON file (e.g. `draft-class-2026.json`) inside this folder so the game can run fully offline without importing the TypeScript dataset.

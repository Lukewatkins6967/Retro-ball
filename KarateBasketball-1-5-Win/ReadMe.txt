       SHUT UP AND SLAM JAM Karate Basketball

                   Version 1.5

Due to a scheduling error, the local rec center has been double booked for a charity basketball event and a karate championship! Luckily Basketballers these days have the moves to please EVERYONE in attendance. 

Its NBA Jam meets Double Dragon... It's SHUT UP AND SLAM JAM KARATE BASKETBALL!

== CONTROLS == 

	- Arrows / Left stick- run around 
	- C / Controller A – Pass/Call for ball 
	- X / Controller X – Shoot/Karate! 
	- Z / Controller B – Dodge
	- Space / Controller Y – Jump
